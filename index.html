This calendar syncs live across your whole team. You only need to do this setup once — then share the link with teammates.

📋 How to get your Firebase config (3 steps)
  1. Go to console.firebase.google.com → Add project
  2. In your project: Build → Realtime Database → Create database (choose test mode)
  3. Project Settings (⚙️) → General → Your apps → Web app (</>) → Register app → copy the config values below
🎉 Connected! Share this link with your team:

Anyone who opens this link will be instantly connected to your shared calendar — no setup needed.

📁 How to host on Google Drive (for persistent team access)
  1. Download this HTML file (use your browser's Save As or right-click the page)
  2. Upload it to Google Drive
  3. Right-click the file → Share → Anyone with the link → Viewer
  4. Open the file in Drive and copy the URL from your browser — that's your team's live link!
  5. Alternatively: Use tiiny.host or netlify drop for instant free hosting
